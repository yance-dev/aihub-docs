# 视频标注

| 空格       | 播放/暂停视频              |
| -------- | -------------------- |
| ←        | 上一帧                  |
| →        | 下一帧                  |
| ctrl + ← | 上一关键帧                |
| ctrl + → | 下一关键帧                |
| K        | 添加/删除关键帧             |
| L        | 断开/连接关键帧             |
| shift+D  | 批量删除关键帧              |
| B        | 打标签                  |
| R        | 【mask识别】绘制正负样本点      |
| U        | 【mask识别】确认轮廓         |
| Y        | 【mask推理】从当前帧向后推理指定次数 |
| Shift+Y  | 【mask推理】从当前帧推理到指定帧   |
| Shift+F  | 【mask推理】从当前帧推理到结束帧   |
| Shift+G  | 【mask推理】从开始帧推理到当前帧   |
| T        | 【框推理】从当前帧向后推理指定次数    |
| Shift+T  | 【框推理】从当前帧推理到指定帧      |
| \[       | 【框推理】从当前帧推理到结束帧      |
| ]        | 【框推理】从开始帧推理到当前帧      |
