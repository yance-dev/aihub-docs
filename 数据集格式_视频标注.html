- videos
  - 1.mp4
  - 2.mp4
- annotations
  - 1.json
  - 2.json

json格式

同LabelFree Video (single json)

AE Video

此格式仅支持导入，不支持导出

目录结构

一个视频对应一个json文件，视频文件和对应的json文件同名

- videos
  - 1.mp4
  - 2.mp4
- annotations
  - 1.json
  - 2.json

json格式

名称 类型 必选 中文名 说明
video_name string true 视频文件名 文件名，默认在videos目录下
label_info [object] true 标注信息 标注信息
» frame_id integer true 帧号 从1开始
» bbox_info [object] true none
»» label string true 标签
»» box [int] false 检测框 [x1,y1,x2,y2]

    "video_name": "123.mp4",  
    "label_info": [
